#  e-Plant Shopping App

A React + Redux Toolkit based shopping cart application for plants.  
Users can browse plants, add items to cart, update quantity, and view total price.

---

##  Features

-  Browse different plant categories  
-  Add plants to cart  
-  Increase quantity  
-  Decrease quantity  
-  Remove items from cart  
-  Live total price calculation  
-  Clear cart option  

---

## Tech Stack

- React JS  
- Redux Toolkit  
- React Redux  
- JavaScript (ES6+)  
- CSS  
- Vite
